#DASH a twist on snake

DASH is a game that was redisgned and recreated using python. It is a twist from the game "Snake".
DASH has a unique twist on the game, rather than using a snake and an apple, DASH makes use of a mouse trying to eat cheese where cats
try and chase the mouse

#Prerequisites
Enusre you have the latest version of python installed (https://www.python.org/downloads/)
Ensure you have access to a code editor, prefferably Visual Studio Code

After downloading DASH you and have met all the requirments you are free to play the game
